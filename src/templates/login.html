Welcome to the Health Portal

A secure cloud-based medical portal for managing healthcare data.


Patients can:

  • Manage/Schedule appointments
  • Transfer files securely
  • View their healthcare provider

Doctors can:

  • View all patient appointments
  • Transfer files securely to all assigned patients.
  • Manage assigned patients

Cloud Services

Through AWS Services & IAC/Terraform, the application provides cloud services for managing healthcare data.


AWS Services

  • EC2 Instances & Target Groups
  • Network Segmentation (VPC & Subnets)
  • Relational Database Service (RDS)
  • Identity and Access Management (IAM)
  • CloudWatch for monitoring
  • S3 bucket for file transfer & Object Storage
  • WAF & ALB for request filtering
